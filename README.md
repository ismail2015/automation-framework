# Hybrid Automation Framework (Web + Mobile)

## Overview
This project demonstrates a professional automation testing framework for both web and mobile applications.

It uses:
- Selenium WebDriver (Web automation)
- Appium (Mobile automation)
- JUnit 5 (Test framework)
- Allure Reports (Test reporting)
- Maven (Build management)
- GitHub Actions (CI/CD)

---

## Features

- Scalable test automation framework
- Page Object Model (POM) design pattern
- Web and mobile test coverage
- Allure reporting with screenshots on failure
- CI/CD integration for automated execution
